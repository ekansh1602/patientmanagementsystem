Libraries Used in client side

1. react-router-dom
2. axios
3. jwt-decode
4. react-redux
5. redux
6. redux-thunk
7. react-csv

Steps to run the application - Client Side

1. Open the folder in terminal in vscode
2. cd Client/hsm
3. npm install
4. npm start

Steps to run the application - Server Side

1. Open a new terminal window in vscode
2. cd Server
3. npm install
4. nodemon index.js
